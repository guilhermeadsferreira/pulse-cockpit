# Frontend Sênior

## Visão Geral

Referência técnica do frontend. Define arquitetura, padrões e caminhos técnicos. Garante qualidade sistêmica, eleva o nível do time e atua como ponto focal da engenharia de interface.

---

## Responsabilidades

- Definir e evoluir a arquitetura frontend: organização de módulos, pastas, domínios e padrões de componentização
- Liderar decisões técnicas de frontend com avaliação explícita de trade-offs
- Garantir qualidade não funcional: performance, acessibilidade, escalabilidade visual e manutenibilidade
- Identificar e reduzir dívida técnica de forma proativa
- Conduzir e evoluir o design system junto com Design
- Criar tech designs (RFCs leves) para funcionalidades complexas
- Influenciar grooming e refinamento com visão técnica de frontend
- Mentorar Frontend Pleno e Júnior via code review, pair programming e direcionamento
- Atuar como ponte técnica entre Frontend, Backend, Design e Produto

---

## Não Responsabilidades

- Gerir pessoas ou conduzir avaliações de desempenho
- Substituir o Tech Lead em decisões de arquitetura cross-stack
- Tomar decisões de UX e produto sem alinhamento com Design e PM
- Ser o único responsável por qualidade de entrega (responsabilidade compartilhada do time)

---

## Interfaces

| Com quem | Natureza da interface |
|---|---|
| Tech Lead | Alinhamento arquitetural e co-decisão em escolhas técnicas de alto impacto |
| Frontend Pleno / Júnior | Mentoria, code review estruturado e elevação técnica do time |
| Design | Evolução do design system; viabilidade técnica de soluções visuais |
| Backend | Contratos de API, estratégias de integração e consistência de dados |
| QA Engineer | Estratégia de testes de frontend, cobertura de edge cases e regressões visuais |
| Produto | Estimativa de complexidade, viabilidade e impacto técnico de funcionalidades |

---

## Competências Esperadas

- Domínio profundo de React, React Native e Next.js (ou equivalentes)
- Arquitetura de frontend: componentização em larga escala, modularização, design system
- Performance avançada: Web Vitals, análise de bundle, renderização SSR/SSG, lazy loading
- Acessibilidade (WCAG básico a intermediário)
- Avaliação e comunicação de trade-offs técnicos
- Escrita de RFCs e documentação técnica
- Comunicação técnica para públicos variados (engenharia, design, produto)
