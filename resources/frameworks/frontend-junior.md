# Frontend Júnior

## Visão Geral

Implementa telas e componentes simples com orientação direta. Foco em aprendizado, aderência ao design system e qualidade visual básica.

---

## Responsabilidades

- Implementar telas e componentes simples seguindo especificações de design e design system
- Garantir responsividade básica conforme padrões do projeto
- Seguir convenções de código, estrutura de componentes e nomenclatura definidas pelo time
- Comunicar bloqueios de design, UX ou técnicos com antecedência
- Participar de code reviews como autor
- Corrigir bugs de interface sob orientação

---

## Não Responsabilidades

- Definir arquitetura de frontend ou design system
- Propor padrões de componentes ou organização de pastas sem direcionamento
- Tomar decisões de UX de forma independente
- Liderar iniciativas técnicas ou mentorear outros

---

## Interfaces

| Com quem | Natureza da interface |
|---|---|
| Frontend Pleno / Sênior | Recebe direcionamento técnico, mentoria e code review |
| Design | Recebe especificações visuais; alinha dúvidas de implementação |
| QA Engineer | Colabora na identificação e correção de bugs visuais e funcionais |

---

## Competências Esperadas

- HTML, CSS e JavaScript fundamentais
- React (ou framework principal do time) — componentes, props, estado básico
- Leitura e uso correto de design system
- Responsividade e boas práticas visuais básicas
- Git e versionamento
- Ferramentas de debug básicas (DevTools, console)
