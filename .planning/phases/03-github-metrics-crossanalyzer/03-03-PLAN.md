**2. Adicionar metodo publico `extractNarrativeContext(slug: string): string`:**
- Le perfil.md do slug via `readFileSync`
- Le PersonConfig via `PersonRegistry.get(slug)`
- Constroi paragrafo narrativo:
  ```
  "{nome} atua como {cargo} ({nivel}) na area {area ?? 'nao especificada'}, na empresa desde {inicio_na_empresa ?? 'data nao registrada'} e na funcao atual desde {inicio_na_funcao ?? 'data nao registrada'}.{promoText}{pdiText}"
  ```
  Onde:
  - `promoText`: se `em_processo_promocao === true` → ` Atualmente em processo de promocao para ${objetivo_cargo_alvo ?? 'proximo nivel'}.` senao vazio
  - `pdiText`: se PDI tem itens ativos (status !== 'concluido') → ` PDI com ${count} objetivo(s) ativo(s).` senao vazio
- Retornar a string (1 paragrafo)
- Se pessoa nao encontrada ou perfil nao existe, retornar string vazia

**3. Adicionar metodo publico `computeBaseline3Months(slug: string): { avgCommits: number; avgPRsMerged: number; avgReviews: number } | null`:**

- Pega os ultimos 3 meses do historico (sort keys descending, slice 0,3)

- Calcula media de cada campo (soma / quantidade de meses que tem dados)
- Se nenhum historico, retornar null
- Arredondar com Math.round
